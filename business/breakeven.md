# breakeven.md

## Unit Economics & Break-even Analysis

### Cost per Active User
- **Compute**: $0.10/user/month
- **Storage**: $0.05/user/month
- **Bandwidth**: $0.02/user/month
- **Total Cost per Active User**: 
  - **Total** = Compute + Storage + Bandwidth
  - **Total** = $0.10 + $0.05 + $0.02 = **$0.17/user/month**

### Pricing Tiers
| Tier          | Price ($/mo) | Features                                                                 |
|---------------|--------------|--------------------------------------------------------------------------|
| Basic         | $5           | Idea capture, basic organization, and prioritization tools              |
| Pro           | $15          | All Basic features + collaboration tools, task assignment, and reporting |
| Enterprise     | $30          | All Pro features + advanced analytics, integrations, and dedicated support |

### Customer Acquisition Cost (CAC) Range
- Estimated CAC: **$50 - $100** per user
  - This includes marketing expenses, sales efforts, and onboarding costs.

### Lifetime Value (LTV) Estimate
- Average Revenue Per User (ARPU):
  - Basic: $5/month
  - Pro: $15/month
  - Enterprise: $30/month
- Average user retention: 24 months
- LTV Calculation:
  - Basic: $5 * 24 = **$120**
  - Pro: $15 * 24 = **$360**
  - Enterprise: $30 * 24 = **$720**

### Break-even Users Count
- Fixed Costs (Monthly): **$2,000**
- Break-even Calculation:
  - Break-even Users = Fixed Costs / (ARPU - Cost per Active User)
  - For Basic Tier:
    - Break-even Users = $2,000 / ($5 - $0.17) = **$2,000 / $4.83 ≈ 414 users**
  - For Pro Tier:
    - Break-even Users = $2,000 / ($15 - $0.17) = **$2,000 / $14.83 ≈ 135 users**
  - For Enterprise Tier:
    - Break-even Users = $2,000 / ($30 - $0.17) = **$2,000 / $29.83 ≈ 67 users**

### Path to $10K MRR
- Target Monthly Recurring Revenue (MRR): **$10,000**
- Pricing Tier Strategy:
  - **Basic Tier**: 
    - Users needed = $10,000 / $5 = **2,000 users**
  - **Pro Tier**: 
    - Users needed = $10,000 / $15 = **667 users**
  - **Enterprise Tier**: 
    - Users needed = $10,000 / $30 = **334 users**

### Summary
To achieve $10K MRR, we can focus on different tiers based on our target market:
- **2,000 Basic users**
- **667 Pro users**
- **334 Enterprise users**

This analysis provides a clear path to profitability and helps in strategizing user acquisition efforts.
